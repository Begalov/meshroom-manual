``meshroom_submit``
===================

not included in binary release

Submit a Graph of processes on renderfarm.

.. list-table:: arguments
    :header-rows: 1

    * - argument
      - description
    * - ``meshroomFile``
      - Filepath to a graph file.
    * - ``--toNode``
      - Process the node(s) with its dependencies.
    * - ``--submitter``
      - Execute job with a specific submitter.
